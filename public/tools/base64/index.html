About Base64 Encoding

Base64 is a binary-to-text encoding scheme that represents binary data in an ASCII string format. It's commonly used to encode data that needs to be transmitted over protocols designed for text data, such as email attachments, data URLs, and API tokens.

Common Use Cases

  • Email attachments (MIME encoding)
  • Data URLs in web development
  • API authentication tokens
  • Configuration file encoding
  • Binary data transmission over text protocols

Character Set

Base64 uses 64 characters:

  • A-Z (26 characters)
  • a-z (26 characters)
  • 0-9 (10 characters)
  • + and / (2 characters)
  • = for padding
