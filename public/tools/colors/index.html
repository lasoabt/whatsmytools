Color Format Information

HEX Colors

Hexadecimal color codes use 6 digits to represent RGB values. Format: #RRGGBB where each pair represents red, green, and blue values from 00 to FF.

Example: #667eea = Red:102, Green:126, Blue:234

RGB Colors

RGB uses decimal values from 0-255 for red, green, and blue components. This is the most common format for screen displays.

Example: rgb(102, 126, 234)

HSL Colors

HSL represents colors using Hue (0-360°), Saturation (0-100%), and Lightness (0-100%). More intuitive for color adjustments.

Example: hsl(227, 75%, 66%)
