About URL Encoding

URL encoding (also called percent-encoding) is a mechanism to encode information in URLs. Special characters are replaced with a percent sign followed by two hexadecimal digits representing the character's ASCII code.

Characters That Need Encoding

  • Space → %20
  • ! → %21
  • @ → %40
  • # → %23
  • $ → %24
  • % → %25
  • & → %26
  • + → %2B

Common Use Cases

  • Form data submission
  • Query parameters with special characters
  • API endpoint parameters
  • Search queries in URLs
  • File names with spaces or symbols
Note: Only URL components (query parameters, fragments) should be encoded, not the entire URL including the protocol and domain.
