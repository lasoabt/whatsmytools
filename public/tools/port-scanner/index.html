About Port Scanning

Port scanning is used to discover which ports are open on a network host. This tool tests connectivity to common service ports using browser-compatible methods.

Common Ports

  • 80: HTTP (Web)
  • 443: HTTPS (Secure Web)
  • 22: SSH (Secure Shell)
  • 21: FTP (File Transfer)
  • 25: SMTP (Email)
  • 53: DNS (Domain Name)
  • 993: IMAPS (Secure Email)

Port Status

  • Open: Port accepts connections
  • Closed: Port rejects connections
  • Filtered: Port doesn't respond (firewall)
  • Timeout: No response within time limit
Important: This tool only scans common, safe ports and uses browser-compatible methods. Results may differ from dedicated network scanning tools due to browser security limitations.
